foaf-a-tron: syntax checker

This is an experiment in using Schematron schemas (via XSLT) to check the syntax of FOAF RDF/XML documents against various proposed syntactic profiles.

How does it work? First we compile ourselves XSLT from the transformation of a FOAF syntactic schema (eg. foaf-basic.sch) using one of the XSLTs supplied with Schematron. In our examples we'll use a transform that generates a checker whose output is HTML. We then use the generated XSLT to test actual FOAF documents, viewing the warnings and diagnostics.

Note that there will be many different kinds of FOAF RDF/XML document; a Schematron schema is a test to see if a FOAF document meets a certain pattern of constraints. Failing to meet those is not a huge problem so long as the document is still in RDF/XML syntax. The RDF Validator can be used to check RDF syntactic validity.
