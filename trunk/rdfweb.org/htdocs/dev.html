RDFWeb Developer Resources

RDFWeb is a work in progress. This page gathers together a variety of resources for developers interested in working with the RDFWeb prototype. The system uses XML and RDF. Some familiarity with these technologies is necessary, for anyone who wants to understand the workings of RDFWeb.

Note: RDFWeb to date has largely been a personal project of Dan Brickley and Libby Miller. We are well aware that the technical documentation is rather sparse. By switching to the new CVS-backed web site we hope to improve this situation. This current document is based on the old RDFWeb home page, which described our two implementations.

RDFWeb Prototypes

We have two implementations in progress: one using Perl and DBM, the other using Java and SQL. Both provide tools to aggregate, store, query and present views of the RDFWeb data graph.

Note: the perl demo is broken currently; the code for merging bNode structures was buggy. We have left a frozen copy of the old 2000-2001 demo running, but please not that it is (a) not updating its index from the Web and (b) liable to behave strangely.

  • Libby's prototype Java/SQL implementation, based on the Inkling system and the SquishQL query language.
  • Libby's implementation includes a codepiction demo based on photo metadata indicating who each photo depicts (see co-depiction write up
  • Notes on using SVG for photo metadata
  • RubyRDF (may rewrite the Perl system in Ruby...)
  • broken/frozen: Dan's original Perl-based implementation. In its day, it was evocative and buggy. Now its just buggy. It exposed an HTML view of an aggregated dataset collected by the scutter indexing tool. Some sample entry points: dan, libby...
  • see also Pipsqueak (javascript testbed for querying this stuff based on Jan Grant's js logic code)
  • Damian Steer's RDFAuthor tool is really handy for authoring RDF (and querying it...)
  • FOAFCorp is an experiment in extending FOAF to represent 'corporate friends of friends', inspired by They Rule. It provides an interesting test case for certain sorts of query.

About the demos

The demos are built by RDFWeb indexers that explore a Web of cross-referenced RDF files, aggregating information from multiple sources.
