CVS access

Experimentally in CVS, fireball.dyndns.org.

To grab a copy of the Web site, a login on the CVS server is needed. Replace 'pldab' with relevant username. Anonymous access is not currently supported.

    export CVSROOT=:ext:pldab@fireball.dyndns.org:/usr/local/cvsroot  
    export CVS_RSH=ssh
    cvs co rdfweb.org
