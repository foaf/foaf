RDFWeb Sitemap

Note: This document should provide a friendly sitemap for the RDFWeb.org site and associated sites and projects. At this stage, everything is still pretty unstructured, and we are using this document as a scratchpad for ideas about how views into the site should be structured.

Site overview

The main Web site for the RDFWeb project is at RDFWeb.org. Various other documents, experiments, prototype installations can be found elsewhere in the Web too. The rdfweb.org is also home to personal Web pages and projects.

We don't have a real sitemap yet, just documents and some tentative categories. This page should list all RDFWeb-related documents though.
