Pipsqueak demo: SQLish query over Jan's prolog stuff

Q: What is this?
A: three forms. top one takes SQL-ish RDF queries against some RDF data (depicted in 3rd form). Second form displays results of queries. You can also see the machine-processable version of the query below, if you care. Change 'uri=' in the query string to run this system against different RDF data files. This demo is part server-side, part client-side. Server deals with retrieving and parsing RDF data files; client takes this parsed data and presents an SQL-ish RDF query interface.

Examples

If you load the xmlhack data, this query works quite well... (paste into query window and compile) 


select ?story, ?title, ?time, ?author, ?url where
    ({http://xmlhack.com/stories/rdf#title} ?story, ?title)
    ({http://xmlhack.com/stories/rdf#time} ?story, ?time)
    ({http://xmlhack.com/stories/rdf#author} ?story, ?author)
    ({http://xmlhack.com/stories/rdf#url} ?story, ?url)
