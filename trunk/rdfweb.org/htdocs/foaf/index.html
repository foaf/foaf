FOAF Project homepage

The Friend of a Friend (FOAF) project is about creating a Web of machine-readable homepages describing people, the links between them and the things they create and do.

This page is being replaced by a new site underway at www.foaf-project.org, and is accompanied by the rdfweb.org site, which has a more technical focus.

Useful links: FOAFNaut, FOAF Explorer (bookmarklet), eikeon.com FOAF Web View, JabFOAF.
