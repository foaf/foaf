FOAF Test Data: A Fictional A-Z

by Dan Brickley $Id: intro.html,v 1.1 2003-02-24 13:36:13 danbri Exp $

This directory contains test data for use by FOAF implementors, and for use in documentation examples. The idea was to create 26 fictional entries, of varying complexity. As I write this there are only a handful of test files, and this wrapper page is just a placeholder so folk who stumble across the data have an idea what they've found.

Issues

  • How to indicate (for FOAF indexing tools that find this data) that they are fictional?
  • Need more internationalized content, names, charsets etc.
