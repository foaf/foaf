CVS/SSH server access

Brief info on access to the rdfweb CVS server.

Using Jim as an example, here is how we set up CVS-over-SSH, noting that the SSH server currently runs on an unfashionable port, so we need a tiny wrapper script, ./fssh to deal with this.

[foafnaut]$ cat ./fssh
#!/bin/bash
/usr/bin/ssh -p 2223 $*

[foafnaut]$ export CVS_RSH='./fssh'
[foafnaut]$ export CVSROOT=:ext:jibberjim@foafnaut.org:/var/lib/cvs
[foafnaut]$ cvs co foafnaut.org/
