Inkling Squish-to-SQL query rewriter

This tool takes a Squish query and transforms it into an SQL query targeted at a certain style of RDF database implemented in SQL. It was written by Libby Miller <libby.miller@bristol.ac.uk> based on a PHP version by Matt Biddulph <matt@picdiary.com>. See the brief howto document for internal details.
