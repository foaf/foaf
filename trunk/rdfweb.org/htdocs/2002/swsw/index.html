Semantic Web South-West is a mailing list for Semantic Web developers in the SW of England (mainly Bristol and Bath). The list is currently hosted by YahooGroups and new subscribers are welcome (please send an intro message once your subscription has been approved). List members from the southwest of any other geographical region are also welcome, provided they don't complain about Bristol-Bath focus and they send an intro message to the list :)

on-topics: Anything to do with Semantic Web, XML, RDF, the Web as a distributed information system. Particularly encouraged: talk announcements (pub lunch proposals, chalk'n'talk, or whatever...), photos and writeups of talks...

See the list archives for more background.
