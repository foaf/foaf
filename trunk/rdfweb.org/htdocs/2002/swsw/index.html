Semantic Web South-West

Semantic Web South-West is a mailing list for Semantic Web developers in the SW of England (mainly Bristol and Bath). The list is currently hosted by YahooGroups and new subscribers are welcome (please send and intro message once your subscription has been approved). Anything to do with SW topics, but particularly meeting announcements (pub lunch, chalk'n'talk, or whatever...) are welcome...

See the list archives for more background.
